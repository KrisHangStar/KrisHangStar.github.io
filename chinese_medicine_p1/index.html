# 肾藏精论

​ 这一部分讲述了很常见的一个问题：对于肾精需不需要节制？这里做了一个很形象的比喻，把肾中藏精之处，比作水井，并引用了《周易》里的话来说明适时清理水井，从井中打水的合理性。
​

夫精，即肾中之脂膏也。有长存者，有日生者。肾中有藏精之处，充满不缺，如井中之水，日夜充盈，此长存者也。其欲动交媾所出之精，及有病而滑脱之精，乃日生者也。其精旋去旋生，不去亦不生，犹井中之水，日日汲之，不见其亏，终年不汲，不见其溢。《易》云：井道不可不革，故受之以革，其理然也。

同时，我们存在于华夏人内心的那个中道的思维仍然不曾改变：过度地耗费肾精，仍然会让我们的井变得枯竭干涸。对于体弱的人，当然得更加节制；对于体强的人，则不能过分放纵，也不宜过分节制。

曰：然则纵欲可无害乎？曰：是又不然。盖天下之理，总归自然。有肾气盛者，多欲无伤；肾气衰者，自当节养。《左传》云：女不可近乎？对曰：节之。若纵欲不节，如浅狭之井，汲之无度，则枯竭矣。曰：然则强壮之人而绝欲，则何如？曰：此亦无咎无誉，惟肾气略坚实耳。但必浮火不动，阴阳相守则可耳。若浮火日动而强制之，则反有害。盖精因火动而离其位，则必有头眩、目赤、身痒、腰疼、遗泄、偏坠等症，其者或发痈疽，此强制之害也。

所以，总的来说，还是需要自然而然地去行事，才是比较恰当的。而当今的人们因为生活方式、环境等等原因，大多应处于井水渐枯的状态，这个时候再去打井水，无非是让这口井的使用期限再变短一点，这显然不是我们所期望的。

# 人参论

​ 徐灵胎开门见山地提出 “先破人之家，而后杀其身者，人参也。” 为什么会有如此的说法呢？原来是人参作为一种比较名贵的中药材，人们过度的迷信，外加医生无脑的选择，让很多家庭最终走向了家破人亡的境地。

乃不察其有邪无邪，是虚是实，又佐以纯补温热之品，将邪气尽行补住。轻者邪气永不复出，重者即死矣。夫医者之所以遇疾即用。而病家服之死而无悔者，何也？盖愚人之心，皆以价贵为良药，价贱为劣药。而常人之情，无不好补而恶攻。故服参而死，即使明知其误，然以为服人参而死，则医者之力已竭，而人子之心已尽，此命数使然，可以无恨矣。若服攻削之药而死，即使用药不误，病实难治，而医者之罪，已不可胜诛矣。故人参者，乃医家邀功避罪之圣药也。病家如此，医家如此，而害人无穷矣！

​ 徐灵胎在简单介绍了人参的功效和正确用法之后，随即提出了人参使用中可能带来的副作用，并且一针见血地点出了乱开人参这一问题的症结所在：患者人愚笨，医者怕担当责任。说到这里，作者曾经在看一位中医的时候，听过这么一段话：“药材的贵重与否就和衣服的好坏一样，虽然价格不一，但是便宜的衣服也并非不能保暖。药也一样，如果能够治好病，那就是好药。” 在这里这句话也真是应景，我们在做出各种各样的决定的时候，有没有仔细的思考过，这个决定对于自我的价值，是资本炒作出的价值，还是真正对于自己个人有益的价值。

其破家之故，何也？盖向日之人参，不过一二换，多者三四换。今则其价十倍，其所服，又非一钱二钱而止。小康之家，服二三两，而家已荡然矣。夫人情于死生之际，何求不得，宁恤破家乎？医者全不一念，轻将人参立方。用而不遵在父为不慈，在子为不孝，在夫妇昆弟为忍心害理，并有亲戚朋友责罚痛骂，即使明知无益，姑以此塞责。又有孝之慈父，幸甚或生，竭力以谋之，遂使贫窭之家，病或稍愈，一家终身冻馁。若仍不救，棺殓俱无，卖妻鬻子，全家覆败。医者误治，杀人可恕，而逞己之意，日日害人破家，其恶甚于盗贼，可不慎哉！

​ 人在生病的时候才会回想起自己身体健康时候的那种愉快和自由的感觉。当大病一场后，由于自我求病的急切，家人对与自己的关心，往往这个时候对于金钱毫不吝啬，唯求能够将自己的病痛治好。如果遇到良医，斟酌利弊，会从病人的疾病、家境等多方面考虑用药；但假若遇到昧着良心去捞黑心钱的人，那么这个时候，自己的身体恐怕是不能够被治好，同时钱包也会被掏空。徐灵胎认为，这种医生的行为，甚至比盗贼还可恨可恶，因为他们是在看不见的地方用刀在插向那些什么也不知道的病人。

# 用药如用兵论

​ 这一篇目是从用药的角度深度阐述医理的思想。古代圣人持中守一，恬淡虚无，饮食节制，只有在必要的时候才会用药，这就和用兵打仗是一个道理，只有万不得已才要战争，因为战争必定有伤亡，服药也会给自己的身体带来损伤。

古人好服食者，必生奇疾，犹之好战胜者，必有奇殃。是故兵之设也以除暴，不得已而后兴；药之设也以攻疾，亦不得已而后用，其道同也。故病之为患也，小则耗精，大能伤命，隐然一敌国也。以草木偏性，攻脏腑之偏胜，必能知彼知己，多方以制之，而后天丧身殒命之忧。是故传经之邪，而先夺其未至，则所以断敌之要道也；横暴之疾，而急保其未病，则所以守我之岩疆也；挟宿食而病者，先除其食，则敌之资粮已焚；合旧疾而发者，必防其并，则敌之内应既绝。辨经络而无泛用之药，此之谓向导之师。因寒热而有反用之方，此之谓行间之术...... 然而选材必当，器械必良，克期不衍，布阵有方，此又不可更仆数也。孙武子十三篇，治病之法尽之矣。

​ 与打仗一样，针对不同的病症使用不同战略谋划、派兵布局才是合理的方法，就连《孙子兵法》中的思想也可以运用到治病的过程中。我想，这就是华夏文明传统文化中伟大的地方：天人合一的思想内涵赋予了这棵大树上各个枝叶统一而华美的精神底蕴。我们能够以一个综合连贯的视角去认识和感受这个世界。
