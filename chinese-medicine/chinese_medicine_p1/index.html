何以学医，以何学医——读《医学源流论》（1）

# 总论

​ 医学，在中国自古以来都被认为是一门天人之学。

​ 医学的产生，自然是为了纠正一个人疾病的状态，那么问题来了，什么样的情况可以被认为是人患病了呢？中医认为，这个判断一个人正常与不正常的标准，是天地的阴阳二气。假如人的身体的气机与天地之气不同步，那么就会导致人身体系统运行的失常，进而导致疾病的产生。由此可以推出，如果人处于天人合一的状态，那么就是最健康的。这个思想也点明了中医治疗身体的最本源的理念：让人与天地和一，按照天地的变化来规律地生活，从而达到治未病的效果。

​ 上古时期，人们对于天地环境的变化并非是同今日一样漠然，所以大多数人在本能地感受到了天地的变化之后，就能够自我纠正并与它保持同步运行。然而，在神农和黄帝的时期，人们慢慢与天地割裂开来了，为了给人们治病，神农写了《神农本草经》，黄帝写了《黄帝内经》。这两部经典是中医理论的源头，也是学习中医必须明白的两本古籍。

​ 而在这篇读书笔记中，我主要记录的是读清代徐灵胎的《医学源流论》的体悟。在这本书中，他以他极高的禀赋和医学经验，记录了各种大大小小的医学中常见的问题。这些问题，即使到了今天，也仍然随处可见。病人和医生都能从这本书中找到医学真正的影子，为自己的求医或者行医之路指明方向。

# 篇目分论

# 医非人人可学论

在徐先生的那个时候，医学就已经沦落为流俗之学了，它被人视为无聊时候的学习消遣，亦或是挣钱养家的一个手段。但是大家没有意识到的是，最初，医学是一门圣人的学问。

乃古圣人所以泄天地之秘，夺造化之权，以救人之死。

如果我们思索到医学的本质，那就得追溯生命的本质，进而追溯到我们这个宇宙的本质。所以医学的最高境界，其实是追究天地的本源，从造化之良能中夺取那个生杀的权利。既然如此，医学不是人人都可以学习，那也合情合理了。具体来讲，需要的能力有哪些呢？在这部分有论述：

能力原因
聪明敏捷医理的复杂性
博学通达古医书文辞的复杂性
虚怀灵变病情变化的迅速
勤读善记经方病症等内容浩如烟海
鉴别判断奇说怪论纷繁

可以看到，为了真正成为一个上医，真的是需要有很强的天赋，以及长期不懈的努力，并有良师引导才行。所以，如果真心要走这条路，非得要有水滴石穿的精神不可！

# 祝由科论 / 用药如用兵论

这几个篇目都是在阐述具体的医理的。对于祝由这一方法，徐先生引用了《黄帝内经》中的篇目来予以解读：

祝由之法，《内经・贼风篇》，岐伯曰：先巫知百病之胜，先知其病所从生者，可祝而已也。又《移精变气论》，岐伯云：古恬憺之世，邪不能深入，故可移精祝由而已。今人虚邪贼风，内着五脏骨髓，外伤空窍肌肤，所以小病必甚，大病必死，故祝由不能已也。

从这里岐伯的看法，我们可以看出，上古之人由于恬淡虚无，邪气不能深入，所以通过祝由这一种类似于心灵感受引导的学问，可以实现治病的效果。但是当今的时代，人们的病已经很深了，所以祝由很难以起作用。

而对于用药，这里开门见山地点明了用药其实是和打仗一样的。那么既然是打仗，就一定会有伤亡，所以最好的做法是能不打就不打，必要时再用兵。从这个角度来看，常常没事有一点小问题就去开一大堆药物，显然是对自己的身体有害的。经常服药的人，必然会生怪病，就如同经常打仗的人，总会遇到奇怪的祸端。而对于不同的病症，就如同战场的情况不同，一定要实用合适的方法。例如经脉中的邪气，我们就应该像断粮草一样切断它的道路；对于突发的横暴的疾病，应该先保住患者的性命，就如同在祸乱时先保护自己的一江一土；对于新旧合发的疾病，应该防止它们进一步和合，就像把战时城中的内应先清理掉一般。总而言之，兵法如用药法，这是习医之人应该深刻领悟的。

我想，这就是华夏文明传统文化中伟大的地方：天人合一的思想内涵赋予了这棵大树上各个枝叶统一而华美的精神底蕴。我们能够以一个综合连贯的视角去认识和感受这个世界，兵法医法一道耳！

# 人参论 / 劫剂论

​ 这两篇，可以说是对世间各种医学乱象的针砭。世事变迁，却发现人性确好像几百年来未曾改变……

​ 徐灵胎开门见山地提出 “先破人之家，而后杀其身者，人参也。” 为什么会有如此的说法呢？原来是人参作为一种比较名贵的中药材，人们过度的迷信，外加医生无脑的选择，让很多家庭最终走向了家破人亡的境地。

乃不察其有邪无邪，是虚是实，又佐以纯补温热之品，将邪气尽行补住。轻者邪气永不复出，重者即死矣。夫医者之所以遇疾即用。而病家服之死而无悔者，何也？盖愚人之心，皆以价贵为良药，价贱为劣药。而常人之情，无不好补而恶攻。故服参而死，即使明知其误，然以为服人参而死，则医者之力已竭，而人子之心已尽，此命数使然，可以无恨矣。若服攻削之药而死，即使用药不误，病实难治，而医者之罪，已不可胜诛矣。故人参者，乃医家邀功避罪之圣药也。病家如此，医家如此，而害人无穷矣！

​ 徐灵胎在简单介绍了人参的功效和正确用法之后，随即提出了人参使用中可能带来的副作用，并且一针见血地点出了乱开人参这一问题的症结所在：患者人愚笨，医者怕担当责任。说到这里，作者曾经在看一位中医的时候，听过这么一段话：“药材的贵重与否就和衣服的好坏一样，虽然价格不一，但是便宜的衣服也并非不能保暖。药也一样，如果能够治好病，那就是好药。” 在这里这句话也真是应景，我们在做出各种各样的决定的时候，有没有仔细的思考过，这个决定对于自我的价值，是资本炒作出的价值，还是真正对于自己个人有益的价值。

其破家之故，何也？盖向日之人参，不过一二换，多者三四换。今则其价十倍，其所服，又非一钱二钱而止。小康之家，服二三两，而家已荡然矣。夫人情于死生之际，何求不得，宁恤破家乎？医者全不一念，轻将人参立方。用而不遵在父为不慈，在子为不孝，在夫妇昆弟为忍心害理，并有亲戚朋友责罚痛骂，即使明知无益，姑以此塞责。又有孝之慈父，幸甚或生，竭力以谋之，遂使贫窭之家，病或稍愈，一家终身冻馁。若仍不救，棺殓俱无，卖妻鬻子，全家覆败。医者误治，杀人可恕，而逞己之意，日日害人破家，其恶甚于盗贼，可不慎哉！

​ 人在生病的时候才会回想起自己身体健康时候的那种愉快和自由的感觉。当大病一场后，由于自我求病的急切，家人对与自己的关心，往往这个时候对于金钱毫不吝啬，唯求能够将自己的病痛治好。如果遇到良医，斟酌利弊，会从病人的疾病、家境等多方面考虑用药；但假若遇到昧着良心去捞黑心钱的人，那么这个时候，自己的身体恐怕是不能够被治好，同时钱包也会被掏空。徐灵胎认为，这种医生的行为，甚至比盗贼还可恨可恶，因为他们是在看不见的地方用刀在插向那些什么也不知道的病人。

​ 至于劫剂，这更是医德沦丧的一个表现。所谓劫剂，就是 “取效于一时，不顾人之生死” 的药方。在这种用药中，医者希望用猛药抑制邪气，由于药效强劲，正气一下生发，将邪气抑制住，显得好像病症好了很多。殊不知邪气复发之时，元气合邪气而发，更加重了邪入身体的程度，造成病人患上更严重的疾病。而且，在当时，这种情况已然不少见了。

此等害人之术，奸医以此欺人而骗财者十之五。庸医不知，而效尤以害人者，亦十之五。为医者可不自省，病家亦不可不察也。

如若出现这种情况，一半都是蓄意而为，谋人钱财。所以作为一个独立的人，要尽量不要让自己生病，平时也最好能了解一些医理，这样才能真正保护自己！
